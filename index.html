Smart Contract Token — ERC-20

Project 03 · Web3 · Blockchain

Implemented a standards-compliant ERC-20 token contract in Solidity as a hands-on exercise in understanding on-chain token mechanics. The contract includes minting controls, transfer logic, and ownership patterns following the OpenZeppelin standard. Deployed to a public Ethereum testnet via Remix IDE using Metamask for transaction signing. The project provided direct insight into the full contract lifecycle — from compilation through deployment and on-chain state verification.
